Returns the last value of an array. Consumes the array before returning.
Throws an Error if the array is empty.
Does not terminate on infinite arrays.

## Type

```ts
type last = <E>(s: Array<E>) => E
```

## Example

```ts
import {last} from 'fnxt/array';

const array = [2, 3, 4];
last(array) // -> 4
```
